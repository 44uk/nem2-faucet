## :fire: Setup

```shell
# set enviroment variables
# * COOKIE_SECRET
# * PORT
# * NETWORK
# * API_HOST
# * API_PORT
# * PRIVATE_KEY
# * MAX_XEM
# * MIN_XEM
# * OPT_XEM
# * ENOUGH_BALANCE
# * MAX_UNCONFIRMED
# * WAIT_HEIGHT
# * RECAPTCHA_CLIENT_SECRET
# * RECAPTCHA_SERVER_SECRET
# or edit .env.development

# install packages
$ npm install

# start app
$ npm start

# or for development
$ npm run dev
```
